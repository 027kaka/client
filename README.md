## 🛠️ 배포 및 실행

`Node.js v14.0.0` 에서 테스트되었습니다.

### 의존성 설치

```sh
yarn install
```

### 개발 버전

```sh
yarn start
```

### 배포 버전 빌드

```sh
yarn build
```

### 배포

```sh
npx serve -s build
```
